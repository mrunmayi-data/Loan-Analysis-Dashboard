# Loan Analysis Dashboard

## Overview
This Power BI dashboard analyzes loan applications, funded amounts, repayment performance, and loan quality metrics.

## Tools Used
- Power BI
- DAX
- Excel

## Key Features
- Total Loan Applications
- Funded Amount Analysis
- Amount Received Analysis
- Good Loan vs Bad Loan Analysis
- Monthly Trends
- Borrower Insights

## Skills
Power BI, Data Modeling, DAX, KPI Reporting, Dashboard Development
